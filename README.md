❄️ MasterFrio – Frontend do Sistema ERP ❄️

Este repositório contém o desenvolvimento do frontend de um Sistema de Planejamento de Recursos Empresariais (ERP) voltado para uma empresa especializada em instalação, manutenção e gestão de informações essenciais relacionadas a sistemas de ar-condicionado.

O objetivo deste ERP é centralizar, otimizar e organizar os processos internos da empresa, incluindo atendimento, ordens de serviço, controle de técnicos, cadastro de clientes e acompanhamento operacional.

- Tecnologias Utilizadas

Next.js (com create-next-app) – Estrutura base do projeto

TSX (TypeScript + JSX) – Segurança de tipos e componentes mais robustos

Tailwind CSS – Design rápido, responsivo e customizável

- Funcionalidades Desenvolvidas (Frontend)

Interface moderna e responsiva

Componentização utilizando TSX

Layout voltado para usabilidade em ambiente operacional

Estrutura escalável para integração com API do ERP

Organização de páginas utilizando o diretório app do Next.js
